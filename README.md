# CS50P Repository

## Overview
- Collection of problem sets, exercises, and projects from CS50P
- Focus on clean, well-structured, and well-documented Python code
- Includes weekly assignments, final projects, and personal extensions

## Structure
- **week_X/**: Weekly problem sets and solutions
- **projects/**: Larger end-of-course projects
- **notes/**: Personal notes and key learning points

## Requirements
- Python 3.x
- Additional libraries may be required depending on the specific folder (see local requirements files)

## How to Run
- Navigate into the desired folder
- Run scripts using: `python file.py`
